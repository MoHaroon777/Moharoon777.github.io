Project Deliverables

This page outlines the required deliverables for the website project, as requested in the provided specification. The website is hosted at the following URL: https://moharoon777.github.io/

Bootstrap Features Used

The following Bootstrap features were used in the creation of this website:

  • Navbar: Used for site navigation, includes a brand logo, toggle button for responsiveness, and navigation links.
  • Grid System: Utilized for layout structure, allowing for responsive design across different screen sizes.
  • Carousel: Used for displaying projects.
  • Buttons: Various styled buttons used throughout the page, including the fixed 'Contact Me' button.
  • Cards: Employed to present information such as work experience and awards in a structured format.
  • Alerts: Used for providing feedback, such as the email confirmation message.
  • Typography: Used Bootstrap's built-in text styling for a consistent appearance.
  • Containers: Used to create responsive fixed-width areas in the page
  • Collapse: Utilized in the navbar to create the hamburger menu for smaller screens.
